About TrP Tools

What is TrP?

TrP is a Roblox game created by OneSkyVed which you can play here

Who are these tools for?

These tools are mainly useful for people running self hosted servers or private servers

What tools are there?

Dispatch portal:

Allows you to easily keep track of assigned routes from the ExportVehicles command

TDW Stage light manager:

Allows you to easily create JSON data for events with the TDW stage

TrP Docs:

Documents everything about using TrP commands and self hosted servers
